Simplest Setup

One command. Everything installs.

Paste this into Terminal on your Mac and walk away. Takes about 10 minutes.

curl -fsSL https://terrapin.tools/install.sh | bash
✓ Installs Ollama and Gemma automatically ✓ Agent starts every time you log in ✓ No terminal needed after setup
Or install manually step by step
Step 1

Download Ollama

Ollama runs AI on your computer. Download it at ollama.com — installs like any app.

Download Ollama →
Step 2

Open Terminal and run one command

Terminal is an app already on your Mac. Press Command + Space, type Terminal, hit Enter.

ollama pull gemma4

This downloads your AI brain — about 5GB, like downloading a movie. Takes a few minutes.

Step 3

Start Terrapin

Open Terminal and run one command:

npm run agent

Open this in your browser and bookmark it: http://localhost:7777/agent.html

Step 4 (Optional)

Set up email sending

Want Terrapin to send invoices and quotes by email? Run this once:

npm run setup-mail

Everything runs on your computer. Nothing goes to the cloud. Your data never leaves your machine.

On Windows? Manual setup only for now — Windows auto-install coming soon.
